# Elm Spotify learning exercise

Set ENV `SPOTIFY_CLIENT_ID` as the Spotify oAuth client id.

Set ENV `SPOTIFY_REDIRECT_URL` as the Spotify oAuth redirect url.

```yarn run dev```
